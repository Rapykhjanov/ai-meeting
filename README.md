# AI Meeting Notes 

AI Meeting Notes — это система, которая превращает голосовые встречи в структурированные, полезные и легко читаемые заметки с помощью AI.

Проект включает Telegram-бота и веб-дешборд для командной работы с встречами, решениями и задачами.

---

##  Проблема, которую решает проект

В большинстве компаний важные решения, договорённости и задачи теряются сразу после встречи.

Этот продукт решает проблему:
- превращает разговоры в структурированные заметки
- фиксирует решения и action items
- помогает командам не терять задачи
- делает всё доступным в одном месте

---

##  Основная идея

Пользователь отправляет аудио или текст в Telegram → AI обрабатывает встречу → система генерирует:

- краткое резюме встречи (3–5 предложений)
- ключевые решения
- задачи (action items) с ответственными
- основные темы обсуждения

Все данные автоматически сохраняются в веб-дешборде.

---

##  Функционал

### Telegram Bot
- Принимает голосовые сообщения и аудиофайлы
- Поддерживает длинные записи (не только короткие voice notes)
- Показывает статус обработки (loading / processing / done)
- Поддерживает текстовые транскрипты
- Позволяет уточнять результат:
  - "сделай короче"
  - "более формально"
  - "только задачи"
- Возвращает структурированный результат, а не “полотно текста”
- Автоматически сохраняет все встречи в систему

---

###  Web Dashboard

Полноценная система управления встречами:

####  Home
- последние встречи
- открытые задачи
- быстрый обзор активности команды

####  Meetings List
- список всех встреч
- поиск по ключевым словам
- фильтрация по датам
- карточки с:
  - датой
  - превью саммари
  - количеством открытых задач

####  Meeting Detail
- структурированный просмотр:
  - summary
  - decisions
  - action items
  - topics
- просмотр транскрипта (с возможностью скрыть/показать)
- редактирование AI-результатов вручную

####  Action Items
- все задачи из всех встреч
- статус-трекинг (open / in progress / done)
- группировка по людям (если имя встречается в разных встречах)

##  Admin Panel (Jazzmin)

Для удобного управления системой добавлена админ-панель на базе **Jazzmin**.

Она используется для:
- управления встречами
- контроля пользователей
- просмотра и редактирования данных
- быстрой диагностики системы

Интерфейс сделан чистым, аккуратным и удобным для администраторов без технического опыта.

## Особенность проекта (дополнительная фича)

Я добавил систему интеллектуальной группировки задач по людям и контексту.

Теперь система может:
- объединять задачи одного человека из разных встреч
- показывать персональные action items
- помогать менеджерам отслеживать ответственность

Это делает продукт ближе к реальному инструменту для команд, а не просто “AI демо”.
